# Contributing

This repository is defensive-only.

Accepted contributions should improve:

- governance quality
- evidence quality
- privacy-preserving correlation
- false-positive reduction
- safe red-team simulation
- adapter clarity
- human-review workflows
- claim-boundary precision

Do not add operational misuse instructions, exploit payloads,
credential-harvesting examples, malware workflows, evasion recipes,
weaponization details, or harmful procedures to tests, docs, examples, fixtures,
issues, or pull requests.

Do not submit confidential, proprietary, export-controlled, classified,
restricted, employer-owned, customer-owned, third-party-owned, offensive,
harmful, or unsafe information.
